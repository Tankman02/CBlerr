# CBlerr Programming Language

CBlerr - это низкоуровневый язык программирования, разработанный для системных задач и написания кода, близкого к железу. Основная цель проекта: создание эффективного инструмента для разработки операционных систем и производительного софта.

## Особенности языка
* **Flux Parser:** Собственная архитектура парсинга для построения абстрактного синтаксического дерева (AST).
* **Мономорфизация:** Поддержка генерации специализированного кода для различных типов данных.
* **Интегрированный отладчик:** Наличие модуля Debugger для анализа работы программы.

## Структура проекта
* `core/` - ядро компилятора.
    * `lexer.py` - лексический анализатор (разбивает код на токены).
    * `flux_parser.py` - синтаксический анализатор.
    * `compiler.py` - основной модуль сборки.
    * `monomorphizer.py` - обработка обобщенных типов.
    * `type_checker.py` - проверка типов.
* `calc.cbl` - пример программы на CBlerr (калькулятор).

## Как компилировать код

Для работы компилятора требуется Python версии 3.10 или выше.

### 1. Подготовка
Убедитесь, что все файлы из папки `core` находятся на своих местах.

### 2. Запуск компиляции
Используйте основной файл `compiler.py`, передав путь к вашему исходному файлу с расширением `.cbl`:

```bash
python build/build.py your_program.cbl
```

Полная документация в: cblerr.netlify.app/docmentation

## Для чего нужен CBlerr
Язык создается как альтернатива существующим решениям для написания системных компонентов. Он сочетает в себе контроль над памятью и удобную структуру описания логики через Flux AST. Это идеальный фундамент для написания ядер ОС, драйверов и высоконагруженных модулей.
